
## Create Table

- creates a a table.

```
CREATE TABLE <tablename> (<columns> <datatype>)
```

## Insert Data

- inserts data into the table.

```
INSERT INTO <tablename> VALUES (<values>)
```

## Select

- selects data based on the condition supplied.

```
SELECT <columns> FROM <tablename> WHERE <conditions>
```

## Delete

- deletes a row or rows based on the condition.

```
DELETE FROM <tablename> WHERE <conditions>
```

## Update

- updates a row or rows based on the condition.

```
UPDATE <tablename> SET <values> WHERE <conditions>
```

## Union

- combines the result of both the queries and produces a single result.

```
<select data from table1> UNION <select data from table2>
```

## Drop

- drops or deletes a table.

```
DROP TABLE <tablename>
```
